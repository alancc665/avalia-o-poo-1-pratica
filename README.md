# Sistema de Biblioteca - POO em C++

## 📝 Descrição
Este projeto é uma atividade prática de Programação Orientada a Objetos. Ele implementa um sistema simples com gerenciamento de usuários e livros, aplicando conceitos de herança e polimorfismo.

## 🛠️ Tecnologias
- Linguagem: C++
- Compilador: GCC/G++

## 🚀 Como Executar
1. baixe o arquivo
2. Compile o código usando o comando: g++ main.cpp -o sistema
3. Execute o programa com: ./sistema
